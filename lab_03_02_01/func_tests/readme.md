# Тесты для лабораторной работы 3, задание 2

## Входные данные
Целые: размер матрицы, матрица

## Выходные данные
Целочисленная матрица

## Позитивные тесты:
- 01 - обычный тест
- 02 - отрицательные числа
- 03 - элемент с минимальной суммой на позиции 0 0
- 04 - элемент с минимальной суммой на позиции 0, m
- 05 - элемент с минимальной суммой на позиции n, m
- 06 - элемент с минимальной суммой на позиции n 0
- 07 - четное количество строк
- 08 - нечетное количество строк
- 09 - четное количество столбцов
- 10 - нечетное количество столбцов
- 11 - размерность 10 10
- 12 - два элемента с минимальной суммой
## Негативные тесты
- 01 - введена стрка вместо количества строк
- 02 - введена строка вместо количества столбцов
- 03 - количество строк > 10
- 04 - количество столбцов > 10
- 05 - количество строк <= 0
- 06 - количество столбцов <= 0
- 07 - строка вметсо элемента массива
- 08 - вещественное число
