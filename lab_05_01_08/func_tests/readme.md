# Тесты для лабораторной работы 5, задание 1

## Входные данные
Целые числа из файла

## Выходные данные
Целое число - расстояние между локальными максимумами

## Позитивные тесты:
- 01 - обычный тест
- 02 - в последовательности 2-ое и предпоследнее число максимумы
- 03 - отрицательные числоа
- 04 - одинаковые по значению максимумы
- 05 - максимумы через 1 значение
## Негативные тесты
- 01 - пустой файл
- 02 - нет максимумов в последовательности
- 03 - только 1 максимум в последовательности
- 04 - не хватает данных для определния максимума
