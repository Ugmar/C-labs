# Тесты для лабораторной работы 10, задание 2

## Входные данные
в зависимости от операции 1 или 2 матрицы, представленные в виде списков ненулвых элементов

## Выходные данные
матрица результат операции

## Позитивыне тесты
- 01 - вывод матрицы
- 02 - вывод матрицы размером 1
- 03 - сложение двух одинаковых матрицы
- 04 - сложение с разным размером матриц
- 05 - умножение квадратных матриц
- 06 - умножение матриц с корректной размером для обычного умножения
- 07 - умножение разных по размеру матриц
- 08 - удаление строки
- 09 - несколько максимальных элементов в матрице при удаление
- 10 - удаление из матрицы 1 на 1

## Негативный тесты
- 01 - отрицательный размер матрицы
- 02 - нулевой размер матрицы
- 03 - вместо элемента матрицы строка
- 04 - неизвестная операция
