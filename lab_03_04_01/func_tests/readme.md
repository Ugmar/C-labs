# Тесты для лабораторной работы 3, задание 4

## Входные данные
Целые: размер матрицы, матрица

## Выходные данные
Целочисленная матрица

## Позитивные тесты:
- 01 - четный размер матрицы
- 02 - нечетный размер
- 03 - матрица 1 на 1
- 04 - матрица 10 на 10
## Негативные тесты
- 01 - введена стрка вместо количества строк
- 02 - введена строка вместо количества столбцов
- 03 - количество строк > 10
- 04 - количество столбцов > 10
- 05 - количество строк <= 0
- 06 - количество столбцов <= 0
- 07 - строка вметсо элемента массива
- 08 - вещественное число
