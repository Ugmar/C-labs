Целочисленный массив

## Позитивные тесты:
- 01 - обычный тест
- 02 - все строки симметричны
- 03 - все строки не симметричны
- 04 - строки с отрицательными элементами
- 05 - строки с 0 элементами

- 08 - количества столбцов > количества строк

- 10 - всего 1 элемент
## Негативные тесты
- 01 - введена стрка вместо количества строк

- 04 - количество столбцов > 10
- 05 - количество строк <= 0
- 06 - количество столбцов <= 0
