# Тесты для лабораторной работы 3, задание 6

## Входные данные
Целые: размер матрицы

## Выходные данные
Целочисленная матрица

## Позитивные тесты:
- 01 - четное
- 02 - нечетное
- 03 - 1 элемент
- 04 - максимальный размер
## Негативные тесты
- 01 - введена стрка вместо количества строк
- 02 - введена строка вместо количества столбцов
- 03 - количество строк > 10
- 04 - количество столбцов > 10
- 05 - количество строк <= 0
- 06 - количество столбцов <= 0
 -07 - разная размерность строк и столбцов
